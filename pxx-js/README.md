## 使用

1. 准备一个nodejs环境 (16.x以上 推荐18.x)
  
2. 通过`npm i`安装依赖
   
3. 运行插件：

终端执行 ——>
# npm run xxx

# 或者 
npm run decode -- -t type [-i input.js] [-o output.js]
```



`type`意思：
* common (高频局部混淆)
* jjencode (sojson.com 版本)
* sojson
* sojsonv7
* obfuscator

1、默认输入文件为`input.js`，文件中不能包含除混淆代码以外的内容（非混淆代码）。

2、默认输出文件为`output.js`。
